# CTF Challenges Portfolio

Welcome to my **CTF Challenges Portfolio**! 🕵️‍♂️  

This repository contains some of the Capture The Flag (CTF) challenges I have completed as part of my cybersecurity learning journey. These challenges showcase skills in **log analysis, metadata forensics, pattern recognition, and problem-solving**.  

---

## 📁 Challenges

### 1. PDF Metadata Forensics
**Challenge:** Hidden flag in a PDF file  
**Tools Used:** `exiftool`, terminal commands  
**Description:**  
- Extracted metadata from a PDF file to uncover hidden flag fragments.  
- Learned how information can be hidden in file properties and how to retrieve it.  
- Skills: Metadata analysis, command-line tools, pattern recognition.  

**Solution Files:**  
- `confidential.pdf` (challenge file)  
- `PDF_metadata_solution.txt` (step-by-step solution)  

---

### 2. Log Hunt
**Challenge:** Hidden flag fragments scattered in server logs  
**Tools Used:** `grep`, `sort`, `uniq`, terminal commands  
**Description:**  
- Filtered server logs to find lines containing `FLAGPART:`.  
- Extracted unique flag fragments and reconstructed the full flag.  
- Practiced handling **noisy data** and reconstructing information from multiple sources.  
- Skills: Log forensics, data extraction, attention to detail, pattern recognition.  

**Solution Files:**  
- `server.log` (challenge log file)  
- `LogHunt_solution.txt` (step-by-step solution)
